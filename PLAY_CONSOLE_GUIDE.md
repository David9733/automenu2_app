# Google Play Console 등록 가이드

## 1. 앱 액세스 권한

### 사용하는 권한 및 설명

#### 필수 권한
- **POST_NOTIFICATIONS** (알림)
  - **사용 목적**: 식사 시간 알림 및 푸시 알림 전송
  - **사용 이유**: 사용자가 설정한 식사 시간 1시간 전에 알림을 받기 위해 필요합니다.

- **SCHEDULE_EXACT_ALARM** (정확한 알림 스케줄링)
  - **사용 목적**: 정확한 시간에 식사 알림을 전송하기 위해 필요
  - **사용 이유**: 사용자가 설정한 식사 시간에 맞춰 정확히 알림을 보내기 위해 사용합니다.

- **USE_EXACT_ALARM** (정확한 알림 사용)
  - **사용 목적**: 정확한 알림 스케줄링을 위한 보조 권한
  - **사용 이유**: SCHEDULE_EXACT_ALARM과 함께 사용하여 정확한 알림 전송을 보장합니다.

- **RECEIVE_BOOT_COMPLETED** (부팅 완료 수신)
  - **사용 목적**: 기기 재시작 후 알림 스케줄 재설정
  - **사용 이유**: 기기가 재시작된 후에도 사용자가 설정한 식사 알림이 정상적으로 작동하도록 하기 위해 필요합니다.

### 권한 사용 설명 (사용자에게 표시될 내용)
"이 앱은 식사 시간 알림 기능을 제공하기 위해 알림 권한이 필요합니다. 사용자가 설정한 식사 시간 1시간 전에 알림을 받을 수 있습니다."
