# What Should I Eat Later?

**이따가 뭐 먹지** - 간단한 질문으로 음식을 추천해주는 모바일 앱

## 📱 앱 소개

직장인들을 위한 빠른 음식 결정 앱입니다. 복잡한 선택지 대신 간단한 질문 몇 개로 딱 맞는 음식을 추천해드립니다.

## ✨ 주요 기능

- **식사 시간 선택**: 아침, 점심, 저녁 중 선택
- **상황 선택**: 혼자, 동료, 애인, 가족 중 선택
- **음주 여부**: 간단한 토글로 선택
- **음식 추천**: 조건에 맞는 음식 1-3개 추천
- **스와이프 제스처**: 음식 카드를 좌우로 넘겨보기
