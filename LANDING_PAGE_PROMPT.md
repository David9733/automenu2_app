# Next.js 랜딩페이지 제작 프롬프트

다음 프롬프트를 사용하여 Next.js 기반 랜딩페이지를 생성하세요.

---

## 프롬프트

Next.js 14 (App Router)를 사용하여 "이따가 뭐 먹지" 모바일 앱의 랜딩페이지를 만들어주세요. TypeScript, Tailwind CSS를 사용하고, 반응형 디자인으로 구현해주세요.

### 앱 정보
- **앱 이름**: 이따가 뭐 먹지
- **앱 설명**: 간단한 질문으로 음식을 추천해주는 모바일 앱
- **타겟 사용자**: 직장인
- **주요 기능**:
  - 식사 시간 선택 (아침, 점심, 저녁)
  - 상황 선택 (혼자, 동료, 애인, 가족)
  - 음주 여부 선택
  - 음식 추천 (조건에 맞는 음식 1-3개)
  - 스와이프 제스처로 음식 카드 넘기기
  - 식사 시간 알림 기능

### 디자인 요구사항

1. **색상 테마**
   - 메인 컬러: #FF6B35 (오렌지)
   - 배경: 그라데이션 또는 깔끔한 흰색/다크 모드 지원
   - 텍스트: 다크 모드 대응

2. **레이아웃 구조**
   - 히어로 섹션: 앱 이름, 간단한 설명, 다운로드 버튼
   - 기능 소개 섹션: 주요 기능 4-5개를 카드 형태로
   - 스크린샷 섹션: 앱 스크린샷 (플레이스홀더 이미지 사용)
   - 다운로드 섹션: Google Play, App Store 다운로드 버튼
   - 푸터: 개인정보처리방침 링크, 연락처

3. **스타일**
   - 모던하고 미니멀한 디자인
   - 부드러운 애니메이션 효과 (fade-in, slide-up 등)
   - 아이콘 사용 (React Icons 또는 Heroicons)
   - 반응형: 모바일, 태블릿, 데스크톱 대응

### 페이지 구조

1. **메인 페이지** (`/`)
   - 히어로 섹션
   - 기능 소개
   - 스크린샷 갤러리
   - 다운로드 CTA
   - 푸터

2. **개인정보처리방침 페이지** (`/privacy`)
   - PRIVACY_POLICY.md의 내용을 HTML로 변환
   - 읽기 쉬운 타이포그래피
   - 목차 네비게이션 (선택사항)
   - 뒤로가기 버튼

### 기술 스택

- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **아이콘**: React Icons 또는 Heroicons
- **애니메이션**: Framer Motion (선택사항)
- **폰트**: Noto Sans KR 또는 Pretendard (한글 최적화)

### 필수 기능

1. **SEO 최적화**
   - 적절한 meta tags
   - Open Graph 태그
   - 구조화된 데이터 (JSON-LD)

2. **성능 최적화**
   - 이미지 최적화 (Next.js Image 컴포넌트)
   - 코드 스플리팅
   - 로딩 상태 처리

3. **접근성**
   - 시맨틱 HTML
   - ARIA 라벨
   - 키보드 네비게이션 지원

### 구현 세부사항

1. **히어로 섹션**
   ```tsx
   - 큰 제목: "이따가 뭐 먹지?"
   - 부제목: "간단한 질문으로 딱 맞는 음식을 추천해드립니다"
   - CTA 버튼: "지금 다운로드" (Google Play, App Store 링크)
   - 배경: 그라데이션 또는 앱 스크린샷
   ```

2. **기능 소개 카드**
   ```tsx
   각 기능마다:
   - 아이콘
   - 제목
   - 설명 (1-2줄)
   - 호버 효과
   ```

3. **개인정보처리방침 페이지**
   - 마크다운을 HTML로 변환
   - 섹션별로 구분
   - 목차 링크 (선택사항)
   - 인쇄 친화적 스타일

4. **푸터**
   - 앱 이름
   - 개인정보처리방침 링크
   - 연락처 (이메일, 전화번호 - 플레이스홀더)
   - 저작권 정보

### 추가 요구사항

1. **다크 모드** (선택사항)
   - 시스템 설정 감지
   - 수동 토글 버튼

2. **다국어 지원** (선택사항)
   - 한국어 (기본)
   - 영어

3. **애니메이션**
   - 스크롤 시 fade-in 효과
   - 버튼 호버 효과
   - 부드러운 페이지 전환

### 파일 구조 예시

```
app/
  layout.tsx
  page.tsx          # 메인 랜딩페이지
  privacy/
    page.tsx        # 개인정보처리방침 페이지
components/
  Hero.tsx
  Features.tsx
  Screenshots.tsx
  DownloadSection.tsx
  Footer.tsx
  PrivacyPolicy.tsx
public/
  images/
    screenshot-1.png
    screenshot-2.png
    ...
```

### 개인정보처리방침 내용

PRIVACY_POLICY.md 파일의 내용을 그대로 사용하되, 다음을 수정:
- [개발자 연락처 이메일을 입력해주세요] → 실제 이메일
- [개발자 연락처 전화번호를 입력해주세요] → 실제 전화번호
- 시행일자 및 최종 수정일을 현재 날짜로 업데이트
