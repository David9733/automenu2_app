# Feature Graphic Image Generation Prompt for ChatGPT

## Prompt for ChatGPT (DALL-E or Image Generation)

Create a feature graphic image for a mobile app called "이뭐먹 - 이따가 뭐 먹지?" (What Should I Eat Later?).

**Image Specifications:**
- Size: 1024px width × 500px height
- Format: PNG or JPEG
- Maximum file size: 15MB
- Style: Modern, clean, and appealing

**App Information:**
- App Name: "이뭐먹 - 이따가 뭐 먹지?" (What Should I Eat Later?)
- Tagline: "Simple questions, perfect food recommendations"
- Target Audience: Working professionals
- Main Color: #FF6B35 (Orange)
- Secondary Colors: White, light orange gradients

**App Features to Highlight:**
1. Meal time selection (Breakfast, Lunch, Dinner)
2. Situation selection (Alone, Colleagues, Partner, Family)
3. Food recommendation based on simple questions
4. Quick decision-making tool for busy professionals

**Design Requirements:**
- Include the app name prominently
- Show food-related elements (icons or illustrations)
- Use the main orange color (#FF6B35) as the primary color
- Modern, minimalist design
- Korean and English text can be used
- Should be eye-catching and professional
- Include visual elements representing: food, time, quick decision

**Visual Elements Suggestions:**
- Food icons or illustrations (Korean food, Japanese food, Western food, etc.)
- Clock or time-related elements
- Clean, modern typography
- Orange gradient background
- Simple, friendly illustrations

**Style:**
- Modern and minimalist
- Professional yet approachable
- Clean design with good contrast
- Suitable for Google Play Store

**Text to Include:**
- App name: "이뭐먹" or "What Should I Eat Later?"
- Tagline: "Simple questions, perfect recommendations" (optional)

**Avoid:**
- Cluttered design
- Too many elements
- Low contrast text
- Unprofessional appearance
